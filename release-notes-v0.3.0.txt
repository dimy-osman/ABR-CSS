## What's New in v0.3.0

### Named CSS Colors (66 classes)
- 22 base CSS colors: black, blue, brown, cyan, gold, gray, green, khaki, magenta, orange, orchid, pink, purple, red, salmon, silver, tan, teal, turquoise, violet, white, yellow
- Classes: .bc-*, .c-*, .brdc-*
- Consistent ABR-U abbreviation algorithm (3-letter words stay as-is)
- Examples: bc-blu, c-red, brdc-tal

### Extended Viewport Units (80 classes)
- Added common percentage values: 17, 20, 25, 33, 40, 50, 60, 67, 75, 80, 100
- Units: dvh, vh, dvw, vw
- Includes !important variants

### Documentation
- Updated ABR-U.md with color table and viewport classes with examples
